Do not estimate work in hours, days, weeks, or months. Estimate in energy expenditure (tokens, agent dispatches, human decision gates). The ORGANVM system operates on an AI-conductor model: human directs, AI generates volume. The bottleneck is human decisions and external dependencies, not developer hours.

**Why:** The user has AI agents ("superpowers") that can execute massive parallel work. A task that takes a solo developer "40 hours" takes one agent session. Calendar time only matters for external deadlines (NLnet April 1, Creative Capital April 2) and external dependencies (grant cycles, reviewer responses, community formation).

**How to apply:** Internal plans use gate-based milestones (what must be true), not time-based milestones (when). The NLnet budget uses hours because NLnet requires it — that is a translation for the funder's model, not the system's model. Never say "this will take 2 weeks" — say "this requires 3 agent sessions and 1 human gate."

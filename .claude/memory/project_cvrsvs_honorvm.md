The governance engine extraction is named **Cvrsvs Honorvm** (`cvrsvs-honorvm`). Named after the Roman *cursus honorum* — the sequential order of public offices. ORGANVM orthography: V for U.

- NLnet proposal: "Cvrsvs Honorvm — Governance Engine for Multi-Repository Open-Source Ecosystems"
- PyPI package: `cvrsvs-honorvm`
- Planned repo: `meta-organvm/cvrsvs-honorvm`
- License: Apache 2.0 (new package; existing organvm-engine is MIT — compatible provenance)

**Why:** NLnet application IRF-INST-001 required a standalone extractable identity separate from organvm-engine. The name carries constitutional weight in ORGANVM's Latin naming system.

**How to apply:** Any future session working on the governance engine extraction, NLnet deliverables, or the standalone package should use this name. The NLnet application draft is at `aerarium--res-publica/applications/nlnet-ngi0-commons-2026/draft.md`.
